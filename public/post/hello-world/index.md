# Hello World!


博客（英语：Blog）是一种在线日记型式的个人网站，借由张帖子章、图片或视频来记录生活、抒发情感或分享信息。博客上的文章通常根据张贴时间，以倒序方式由新到旧排列。

任何代码程序学习的第一句永远都是hello world，以帮助初学者更快的建立对语言的基础认识和建立信心。
比如：
```rust
fn main() {
  println!("Hello World!");
}
```

或者是C:
```C
#include <stdio.h>
int main() {
  print("Hello World!");
}
```

或者是更简单的Python:
```Python
print('Hello World!')   #它甚至只需要一行代码！
```

亦或者是HTML：
```HTML
<html>
  <head>
    <title>Hello World!<title>
  </head>
  <body>
    <h1>Hello World！</h1>
  </body>
  <html>
```
